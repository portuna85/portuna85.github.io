## 배열(1차원, 2차원, String 배열, 다차원 배열)

### 배열이란? 같은타입의 데이터를 하나로 묶어서 다루는것.

### 1차원 배열

#### 배열의 선언
 - int[] arr = new int[3]; 또는 int[] arr = {1,2,3}으로 선언과 동시 초기화.

```java
public class Array {
  public static void main(String[] args) {
    int[] arr1 = new int[3];
    int[] arr2 = {1, 2, 3};

    arr1.length;                        // int배열 타입의 arr1 길이.
    System.out.println(arr1);           // arr1의 주소출력
    for (int i = 0; i < arr1.length; i++) {
      System.out.println(arr1[i]);              // arr1[i] arr1배열의 i번째 주소 참조하여 값 출력
      System.out.println(Arrays.toString(arr1));// arr1[i] arr1배열의 i번째 주소 참조하여 값 출력
      System.out.println();
      System.out.println(arr2[i]);              // arr2[i] arr2배열의 i번째 주소 참조하여 값 출력
      System.out.println(Arrays.toString(arr2));// arr2[i] arr2배열의 i번째 주소 참조하여 값 출력
    }
  }
}
```

### 배열을 주소값을 추가할 경우

### 2차원 배열

#### int[][] arr = new int[3][3]; 또는 int[][] arr = {{1,2,3}, {4,5,6}, {7,8,9}};으로 선언과 동시 초기화.

```java
public class Array {
  public static void main(String[] args) {
    int[][] arr1 = new int[3][3];
    int[][] arr2 = {{1, 2, 3},
      {4, 5, 6},
      {7, 8, 9}};

    // int[i][j] 타입의 arr1 i의 배열 길이
    int length = arr1.length;
    System.out.println(arr1);                     // arr1의 주소 값 출력

    for (int i = 0; i < arr1.length; i++) {
      for (int j = 0; j < arr1[i].length; j++) {
        System.out.println(arr1[i][j]);            // arr1[i][i] arr1 2차원 배열의 i행 j열 주소참조하여 값 출력 
        System.out.println(arr2[i][j]);            // arr2[i][i] arr1 2차원 배열의 i행 j열 주소참조하여 값 출력
      }
    }
  }
}
```

### String 배열

#### String[] str = new String[]; 또는 String[] str = ""; 으로 선언과 동시 초기화.

```java
public class ArrayStr {
  public static void main(String[] args) {
    String[] str1 = new String[3];
    String[] str2 = {"Hello", "JAVA"};

    System.out.println(str2[0] + str2[1]);           // String[](String 배열) 타입의 주소값 참조하여 Hello, JAVA 출력
  }
}
```
